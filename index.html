HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Src is the tag used for specifting the source of an image to be displayed
  • The term "tag" or "element" is used interchangably by developers however "element" refers to a whole section of html, opening and finishing tags and everything between whereas "tag" refers to the start and finish tags either end of the element

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Asigning CSS colors can be done using semantic references such as "gray" or "blue", or using hexadecimal code, or hex #, or using RGB values
  • Class selectors can be added to HTML code and then referenced in CSS to style-ise the class selectors used in the HTML

Git

  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a name container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Declaring a function in JS code creates a reusable block of code that performs a specific task.
  • To call a function, we use the name of the function in our JS file where we want the code to execute.
